# Easy Survey

A survey App created with **ReactJs** and **NodeJs**

**Server** side:
- Nodejs;
- Express;
- PassportJs;
- Sendgrid;
- Mongoose;
- Stripe

**Client** side:
- ReactJs;
- Redux;
- Redux-form;
- Redux-thunk;
- React-router;
- Axios;
- Material-ui;
- Materialize-css

**Live** on https://lit-spire-23247.herokuapp.com/
